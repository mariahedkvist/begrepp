Github Flow

är ett arbetsflöde som används med plattformen Github för att underlätta när flera personer arbetar på samma projekt. Genom att jobba på olika saker i olika branches, och namnge sin branch smart, så blir det tydligt vem som gör vad och vilken del av koden som ändras och varför. Exempelvis om en person i teamet har i uppgift att skapa sidans footer, så döper denne rimligtvis sin branch till "footer" eller liknande.
I korthet så består Github flow av följande steg:
Skapa en branch med lämpligt namn > Gör tänkta ändringar > Committa och pusha upp branchen till Github > Skapa ett Pull request och be ev. någon gruppmedlem/projektledare granska, innan denne > Godkänner och tillåter dig att göra en merge till main.

Flexbox

är en typ av layout som är tänkt att göra det enklare att skapa responsiva hemsidor. Det bygger på att man har sk "flex containers", alltså behållare som kan ordnas horisontellt eller vertikalt, storleksanpassas och placeras på ett sätt som gör hemsidan flexibel för olika skärmstorlekar.

DOM (Document Object Model)

är en modell för en hemsidas struktur som gör att man kan manipulera den med ett programmeringsspråk, t.ex. Javascript. Med Javascript kan man t.ex. hämta ett element och lägga till funktioner, eller ändra dess innehåll och styling. Man kan också skapa helt nya element och "skicka" dessa till HTML-sidan. Med DOM kan man se dokumentet med en trädstruktur.

"Fin kod"

innebär att koden är lättläst och förståelig för andra utvecklare. Att man exempelvis tar för vana att döpa sina variabler på ett begripligt sätt, att strukturen i koden är logisk och lätt att följa, och att koden har kommentarer där det skulle kunna uppstå missförstånd eller där man behöver vara extra tydlig.

Ramverk

är ett sätt att hämta kod som någon annan gjort och använda i sitt eget projekt. Det är vanligt att man plockar ihop en "grund" till sitt projekt utifrån några ramverk, och därmed inte behöver börja från noll. Två exempel är Bootstrap, utvecklat av Twitter, och React, utvecklat av Facebook. Båda bygger på Javascript och båda syftar till att enkelt bygga responsiva användarmiljöer.
